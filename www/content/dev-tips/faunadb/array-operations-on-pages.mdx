  When using @faunadb's FQL, the Page type that is returned
  from Paginate() calls can be used in array methods.
title: MDXProvider Dot Notation
---

```haskell
Map(
  Paginate(Match(Index("all_customers"))),
  Lambda(
    "X",
    Select(["data", "firstName"],
      Get(Var("X"))
    )
  )
)
```
