---
title: Consuming private gpr packages with Yarn
tweet:
  To install an npm package from the @github package
  registry using yarn, create the following .npmrc file with
  a personal github access token that has the read:packages
  and repo scopes
---

```
@christopherbiscardi:registry=https://npm.pkg.github.com
//npm.pkg.github.com/:_authToken=PERSONAL_ACCESS_TOKEN
```
