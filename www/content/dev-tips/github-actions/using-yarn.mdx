---
title: Using Yarn
tweet:
  Yarn comes pre-installed when you use the
  `actions/setup-node` GitHub action. No extra work required
---

```yaml
name: Test Package

on: [push]

jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v1
      - uses: actions/setup-node@v1
        with:
          node-version: 12
      - run: yarn
      - run: yarn test
```
