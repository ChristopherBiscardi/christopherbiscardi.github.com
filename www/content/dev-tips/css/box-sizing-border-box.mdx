tweet:
  box-sizing border-box includes the padding and border in
  width calculations. To support legacy components, we can
  use this snippet and set box-sizing content-box on the
  legacy component root.
title: box-sizing border-box
---

```css
html {
  box-sizing: border-box;
}
*,
*:before,
*:after {
  box-sizing: inherit;
}
```
