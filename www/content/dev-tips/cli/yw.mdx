---
tweet: Tired of typing out `yarn workspace <workspace name> <script>`? Try out this alias that uses jq and fzf to let you choose a workspace from a list. `yw develop`
title: yarn workspace alias
---

```sh
yw() {
  yarn workspace $(yarn workspaces info --json | jq '.data' -r | jq "[keys][0] []" -r | fzf) $@
}
```
