---
tweet: In @gatsbyjs, you can fix nested types (like Frontmatter) by specifying the field on the root type
title: fix nested types
---

```js
exports.createSchemaCustomization = ({ actions }) => {
  createTypes(`
    type MarkdownRemark implements Node {
      frontmatter: Frontmatter
    }
    type Frontmatter {
      tags: [String!]!
    }
  `);
};
```
